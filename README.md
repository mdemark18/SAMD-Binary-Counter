# SAMD Binary Counter
PCB by:  Bryce Hill
Code By: Miller DeMark
## Overview
This repository contains the code for the SAMD Binary Counter created for Montana Tech's Electrical Engineering Department for student outreach. 
The counter displays a count from 0-1024 on a Bar LED, and can be communicated with via Serial in a terminal over USB. The following code has comments explaining how it works,
and there are documents going into further depth on how to adjust code as needed or if changes will need to be made in the future.
