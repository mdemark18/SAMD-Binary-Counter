# Microchip Studio (Atmel)

To edit the code for this project, you'll need to download the latest version of Microchip Studio, formerly known as Atmel Studio. Download the package for SAMD microcontrollers and you'll be good to go.<br>

You'll also need to download a copy of this repository.<br><br>
[Microchip Studio](https://www.microchip.com/en-us/tools-resources/develop/microchip-studio#Downloads)
## Getting Started

To open the project, you can do it two ways, either through Microchip Studio itself, or through your File Explorer
